## Require softwares

This project was developed using the following tools: JDK 1.8, Apache Netbeans 13, Apache Tomcat 10, SQL Server 19, Visual Paradigm (for drawing diagram)

## Components

- SWP391-TestDB-Version-4.0.sql: A empty database for testing
- libraries: A folder contains all .jar files libraries used in this project
- Documentation: All documentations generated during project development
- Images: Contains diagram images (diagram too big to fit in the documentation)
